ParmEd

OpenMM Code Samples

ParmEd implements a Python API for using parameter, topology, and coordinate files from either CHARMM or Amber to run a GPU-accelerated molecular dynamics simulation using OpenMM.

The Amber file interface presented here gives the same interface as the one provided by OpenMM, but the AmberParm class here is compatible with the ParmEd API and exposes a more comprehensive interface for modifying the Hamiltonian and underlying parameter set.

The CHARMM file interface included in ParmEd is not currently available in the release versions of OpenMM (version 6.0 as of April 22, 2014). The core routines required to create an OpenMM System object have been ported to OpenMM and will be available in later versions, but developments allowing the structure and parameters to be modified will be implemented here.
